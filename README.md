# Задание 1

## Задача 1

### Текст задачи

Получить список из последних цифр чисел, содержащихся в исходном списке.

### Алгоритм решения

1.Запросить у пользователя количество элементов списка.

Проверить корректность ввода количества элементов с помощью readSize.

Считать введённую строку. Попытаться преобразовать её в целое число. Если преобразование невозможно, вывести сообщение об ошибке и повторить ввод. Если число меньше нуля, вывести сообщение об ошибке и повторить ввод. Если ввод корректный, сохранить значение размера списка.



2.Запросить у пользователя ввод элементов списка.

Проверить корректность каждого введённого элемента с помощью readElement.

Считать введённую строку. Попытаться преобразовать её в вещественное число. Если преобразование невозможно, вывести сообщение об ошибке и повторить ввод. Если ввод корректный, вернуть введённый элемент.



3.Сформировать список элементов с помощью createList

Если размер списка равен нулю, вернуть пустой список. Иначе считать элемент списка. Добавить введённый элемент в список. Рекурсивно сформировать оставшуюся часть списка.



4.Для каждого элемента списка получить последнюю цифру. Преобразовать элемент в строку. Удалить знак минус. Удалить точку. Взять последний символ строки. Преобразовать этот символ обратно в число.

Применить функцию получения последней цифры convertintString ко всем элементам списка с помощью List.map и сформировать новый список последних цифр.



5\. Вывести на экран исходный список и список последних цифр.

### Тестирование

Скриншоты результата работы программы

3.Вводим цифру для проверки окончания, используется функция readDigit, она гарантирует, что введена цифра от 0 до 9

внутри неё применяется List.fold

если элемент заканчивается на нужную цифру

он добавляется к аккумулятору

иначе пропускается

начальное значение аккумулятора 0.0



5\.Вывести на экран исходный список  и результат
